# Common Recording Proprietary System

## Overview
This is a common recording system designed to facilitate recording processes in various setups.

## Setup Instructions
### EndeavourOS/Linux Setup
1. Update your package index:
   ```bash
   sudo apt update
   ```
2. Install Python 3.11:
   ```bash
   sudo apt install python3.11 python3-pip
   ```
3. Install Streamlit:
   ```bash
   python3.11 -m pip install streamlit
   ```

### Running the Application
To run the application, use the following command:
```bash
streamlit run main.py
```

### LAN Access
Ensure that your firewall settings allow LAN access to your application, allowing others on your network to connect to the running Streamlit app.
